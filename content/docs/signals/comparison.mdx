Usual comparison operators can apply to build [conditions](../condition) from expressions.

### Examples

import { Tab, Tabs } from "fumadocs-ui/components/tabs";

```txt tab="greater than"
AAPL > sma(AAPL, 20)

rsi(AAPL, 14) > 80
```

```txt tab="less than"
AAPL < sma(AAPL, 20)

rsi(AAPL, 14) < 20
```
