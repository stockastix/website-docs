Usual mathematical operations can apply to [expressions](./expression).

### Examples

import { Tab, Tabs } from "fumadocs-ui/components/tabs";

```txt tab="Addition"
AAPL + MSFT
```

```txt tab="Substraction"
AAPL - sma(AAPL, 200)
```

```txt tab="Multiplication"
AAPL * AAPL{Volume}
```

```txt tab="Division"
AAPL / MSFT
```
