## Stocks

### Examples

- Apple Inc. historical data

```
AAPL
```

By default, a symbol will return the closing value. You can specify other values:

import { Tab, Tabs } from "fumadocs-ui/components/tabs";

```txt tab="Open"
AAPL{Open}
```

```txt tab="High"
AAPL{High}
```

```txt tab="Low"
AAPL{Low}
```

```txt tab="Close"
AAPL{Close}
```

```txt tab="Volume"
AAPL{Volume}
```
